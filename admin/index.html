When you connect a custom domain:
  1. First add the domain to the project in Vercel (Settings → Domains) and follow the DNS steps
  2. Once the domain is live, change the address here (e.g. https://www.darrensgarage.co.uk)
  3. Saving rewrites every page's SEO data (canonical links, social previews, business schema, sitemap) in one go — live in about a minute
How to get your GA4 Measurement ID:
  1. Go to analytics.google.com and sign in with the business Google account
  2. Admin (gear, bottom-left) → Create Property → name it "Darrens Garage Services", country UK, currency GBP
  3. Choose Web as the platform and enter the site address
  4. Open the created Data Stream and copy the Measurement ID — it looks like G-ABC12DE3FG
  5. Paste it below and save — tracking starts within a few minutes
How to verify with Google Search Console:
  1. Go to search.google.com/search-console → Add property → choose URL prefix and enter the site address
  2. Easiest: if Google Analytics is already set up above, pick the Google Analytics verification method — done, no token needed
  3. Otherwise pick HTML tag: you'll see <meta name="google-site-verification" content="ABC123...">
  4. Copy only the content value (the ABC123... part) and paste it below, save, wait a minute, then click Verify in Search Console
How to verify with Bing Webmaster Tools:
  1. Go to bing.com/webmasters and sign in
  2. Easiest: choose Import from Google Search Console — Bing copies the verified site automatically, no token needed
  3. Otherwise: Add site manually → verification method HTML Meta Tag: you'll see <meta name="msvalidate.01" content="ABC123...">
  4. Copy only the content value, paste it below, save, wait a minute, then click Verify in Bing
